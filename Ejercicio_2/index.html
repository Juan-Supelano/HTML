Introducción a HTML

¿Qué es HTML?

HTML es el lenguaje estándar para crear páginas web. HTML describe la estructura de una página web usando una serie de elementos.

Elementos básicos de HTML

Un documento HTML típico incluye varias etiquetas que definen el contenido y la estructura de la página.

Algunos de los elementos más comunes en HTML son:

<h1> a <h6> - Encabezados, donde <h1> es el más importante y <h6> el menos importante.

<p> - Define un párrafo.

<a> - Define un enlace.

<img> - Inserta una imagen.
